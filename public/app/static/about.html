What is Peerchain ?

Peerchain is a database that stores data about the Peercoin network. Users can inquire for data by entering a block height, a block hash, a transaction hash, or an address.

What is Peercoin ?

Peercoin is a crypto-currency, a decentralized network allowing peer-to-peer financial transactions worldwide over the internet. Through an innovative minting algorithm, the Peercoin network consumes far less energy, maintains stronger security, and rewards users in more sustainable ways than other cryptocurrencies.

What is CDA?

CDA is an abbreviation for Coin Days Accumulated. This is a measurement of how much coin age (measured in days) an addresses' unspent outputs have accumulated/generated. CDA is derived from the formula, (current timestamp - transaction timestamp of unspent output)/86400. CDA also has a cap limit of 90 days,so If an unspent output has been sitting for more than 90 days, the network stops counting the coin age after 90 days.

What is CDA for?

CDA is for those that are interested in minting new coins. Minting is the process of solving a block in the Peercoin network using an addresses' coin age as "hashing power", and receiving a reward in PPC relative to the amount of coin age accumulated. Note that in terms of a minting reward, the amount of coin age that determines how much PPC an address receives as a reward has no cap limit.

What is CDD?

CDD is an abbreviation for Coin Days Destroyed. This is a measurement of how much coin age (measured in days) is destroyed/consumed in a transaction, by the inputs of the transaction. Although CDD serves no real purpose, it is meant as a reference regarding how long it has been since the input addresses' unspent output (now used as an input in the transaction) have been sitting. Note that CDD does not apply to the coinbase transactions in Proof -of- Work blocks.
